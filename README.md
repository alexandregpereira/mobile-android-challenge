
# AMARO Android Challenge
In this exercise, the proposed challenge is to create a catalogue app that displays [AMARO](https://amaro.com/)'s all-time _best-sellers_.

## Technologies
- [Kotlin coroutines](https://github.com/Kotlin/kotlinx.coroutines)
- [AndroidX](https://developer.android.com/jetpack/androidx)
- [ConstraintLayout](https://developer.android.com/training/constraint-layout)
- [Data Binding](https://developer.android.com/topic/libraries/data-binding)
- [JUnit](https://developer.android.com/studio/test)
- [Goblin Adapter](https://github.com/alexandregpereira/goblin-adapter/tree/2.0.0-alpha2)
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture)
- [Retrofit](https://square.github.io/retrofit/)
- [Glide](https://github.com/bumptech/glide)
